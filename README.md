# 方案文件修改助手 (proposal-editor)

专为国内体制内场景设计的方案文档修改AI技能。

## 定位

适用于政府机关、国有企业、事业单位等体制内场景下的公文方案修改，包括：
- 项目建议书
- 可行性研究报告
- 项目计划书
- 预算方案
- 其他正式方案文档

## 核心功能

1. **逻辑联动检查**：修改某处后，自动扫描全文找出基于该内容衍生的其他表述，同步调整
2. **数据一致性**：确保数字、口径、计算逻辑全文统一
3. **多色高亮标记**：按修改类别分配不同颜色，方便用户区分主修改与关联修改
4. **WPS/Word兼容**：支持doc/docx格式

## 适用人群

- 需要修改项目方案文件的国企/政府办公人员
- 需要保持方案逻辑严谨性的咨询顾问
- 需要批量处理方案文档的项目管理者

## 使用方式

在支持WorkBuddy/Claude Code/OpenCLAW的AI助手中加载此skill即可使用。
